

## Changes since version 1.0.52

New Features
- Duplicate Tab: Right-click any tab to duplicate it (same URL, positioned right after original, stays in same group)
- Auto-group new tabs: New tabs (Ctrl+T, links) automatically join the active tab's group
- Cmd/Ctrl+click on pinned sites: Opens in a new tab instead of switching to existing
- Full browser state backup: Extends pinned sites backup to include complete browser state

  
## Description

Arc Browser like Pinned Sites, Bookmarks and Active Tabs in Side Panel

Brings Arc Browser's sidebar experience to Chrome. Displays pinned sites, bookmarks, and active tabs in a vertical side panel.

FEATURES

Pinned Sites (Arc-style Favorites)
• Icon bar at top of panel for frequently used sites
• Custom icons and colors
• Opens as Chrome pinned tab (reuses existing tab if open)
• Export/Import as JSON

Bookmarks
• Tree view with drag-and-drop organization
• Sort by name or date
• Pin bookmarks to the pinned bar

Active Tabs
• All tabs in current window with drag-and-drop reordering
• Sort by domain
• Audio indicator for tabs playing sound
• YouTube chapters listing support  (added for my own YouTube binge-watching needs)

Tab Groups
• Full Chrome tab group support
• Drag tabs between groups
• Reorder groups
• Rename and change group colors

Keyboard Shortcuts
• Toggle sidebar: Cmd+Shift+E (Mac) / Ctrl+Shift+E (Windows/Linux)

Settings
• Font size and icon size
• Hide "Other Bookmarks" folder
• Dark/light mode (follows system)

Permissions
• sidePanel, bookmarks, tabs, tabGroups: Core functionality
• storage: Save pinned sites and settings locally
• favicon: Display website icons
• scripting + youtube.com: YouTube chapter integration

Open Source
Source code available on GitHub (https://github.com/monemone-org/ChromeSideBar).
